📄 Skin Disease Classification using Deep Learning
📌 Project Overview

This project develops a Deep Learning based Skin Disease Classification system using Convolutional Neural Networks (CNN) with MobileNetV2 Transfer Learning.

The model analyzes skin images and predicts the type of skin disease. This system can support early detection and assist dermatologists in diagnosing skin conditions.

🎯 Objective

Build an AI model to classify different skin diseases from images

Improve model performance using data augmentation

Use transfer learning to leverage pretrained CNN models

🧠 Model Architecture

The project uses TensorFlow / Keras with MobileNetV2 for feature extraction.

Key features:

Transfer Learning with MobileNetV2

Data Augmentation

Class imbalance handling

Model checkpoints for best model saving
- Dataset located at: `/content/drive/MyDrive/Skin_Diseases_Classification`
- Classes:
  - Eczema
  - Rosacea
  - Vitiligo
  - Psoriasis
  - Urticaria Hives
  - Athlete’s Foot
  - Acne
